# 🖼️ Avatar Gallery

Расширение для SillyTavern, которое позволяет загружать несколько аватаров для персонажей и персон, и переключать их прямо из чата.

## Идея

Автор идеи — **девушка из комьюнити**. Реализация — Smokshi.

## Возможности

- **Галерея аватаров** — загружай несколько аватарок для каждого персонажа и персоны
- **Зум из чата** — клик по аватару в чате открывает увеличенный просмотр
- **Стрелки ◀ ▶** — переключай аватары прямо в просмотрщике (или клавишами ← →)
- **Лента миниатюр** — внизу просмотрщика видны все загруженные варианты
- **Установка аватара** — кнопка «★ Установить» применяет выбранный аватар к чату
- **Управление в настройках** — добавление, просмотр и удаление аватаров в панели расширения

## Установка

1. Скопируй папку `avatar-gallery` в:
   ```
   SillyTavern/public/scripts/extensions/third-party/
   ```
2. Перезагрузи SillyTavern
3. Расширение появится в панели — раздел **«🖼️ Avatar Gallery»**

## Как пользоваться

1. Открой чат с персонажем
2. В настройках расширения (🖼️ Avatar Gallery) нажми **+** рядом с нужным персонажем или персоной
3. Загрузи одну или несколько картинок
4. Кликни на аватар персонажа в чате — откроется зум-просмотрщик
5. Листай стрелками, нажми **★ Установить** чтобы сменить аватар

## Хранение

Аватары хранятся в настройках расширения (extensionSettings) в формате base64. Изображения автоматически сжимаются до 512px, миниатюры до 80px.

## Совместимость

Работает с любыми персонажами, группами и персонами. Не конфликтует с другими расширениями.
