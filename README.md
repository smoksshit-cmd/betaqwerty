# 🧠 Трекер памяти фактов (Facts Memory Tracker) v2.0

Расширение для SillyTavern, которое автоматически извлекает важные факты из истории чата и хранит их компактно — вместо тяжёлой истории в промпт попадает только нужное.

## Что нового в v2.0

- **Новый дизайн** — полностью переделанный интерфейс
- **Панель сверху** — трекер выезжает сверху по центру экрана (вместо боковой панели)
- **Карточки-плитки** — факты отображаются в grid-layout (как Trello/Pinterest)
- **Круглый виджет** — FAB-кнопка в виде круглого bubble с бейджем-счётчиком
- **Красивые кнопки** — pill-style кнопки с плавными анимациями
- **Адаптация к темам** — сохранена полная поддержка ST CSS-переменных
- **Вся логика сохранена** — сканирование, инъекция, маркеры, флешбеки, API, экспорт/импорт

## Установка

1. Скопируй папку `facts-memory-tracker` в:
   ```
   SillyTavern/public/scripts/extensions/third-party/
   ```
2. Перезагрузи SillyTavern
3. Расширение появится в панели настроек — раздел **«🧠 Память фактов»**

## Возможности

### 🔍 Извлечение фактов
- **Ручной скан** — AI проанализирует историю чата
- **Авто-скан** — каждые N сообщений (настраивается)
- **Авто-маркер** — модель пишет `[FACT: текст | категория]` → факт добавляется мгновенно
- **Скан диапазона** — выбери конкретный участок чата для сканирования

### 📂 Категории
| Категория | Что хранит |
|-----------|-----------|
| 👤 Персонажи | Имена, роли, черты, отношения |
| 📅 События | Ключевые события, решения |
| 🔒 Секреты | Тайны, скрытые мотивы |
| 🌀 Флешбэки | Воспоминания, травмы, образы из прошлого |

### ⚡ Флешбек-триггер
Заставляет персонажа вспомнить факт прямо в следующем ответе.

### 💉 Инъекция в промпт
Компактный блок фактов с настраиваемым порогом важности и лимитом.

### ✏️ Управление
- Inline-редактирование кликом на карточку
- Смена категории и важности через дропдауны
- Отключение без удаления
- Поиск и фильтрация по тексту/категории/важности
