# 🖼 Avatar Gallery

Расширение для SillyTavern, которое добавляет галерею альтернативных аватарок для персонажей и персон.

## Возможности

- **Галерея аватарок** — загружай несколько аватарок для каждого персонажа и персоны
- **Зум из чата** — кликни на аватарку в чате → откроется увеличенная версия
- **Стрелки ◀ ▶** — листай все аватарки прямо в увеличенном виде
- **Установить аватар** — нажми кнопку в зуме → аватарка станет основной
- **Управление в настройках** — сетка превью, добавление и удаление
- **Работает для обоих** — персонажи (боты) и персоны (пользователь)

## Установка

1. Скопируй папку `avatar-gallery` в:
   ```
   SillyTavern/public/scripts/extensions/third-party/
   ```
2. Перезагрузи SillyTavern
3. В настройках расширений появится раздел **«🖼 Avatar Gallery»**

## Как использовать

### Добавить аватарки
1. Открой настройки расширения
2. В секции «🤖 Персонаж» или «👤 Персона» нажми **+**
3. Выбери одно или несколько изображений

### Сменить аватар из чата
1. Кликни на аватарку персонажа или свою в чате
2. Откроется увеличенная аватарка
3. Листай стрелками ◀ ▶ или клавишами ← →
4. Нажми **«Установить аватар»** чтобы применить

### Клавиши
- **← →** — листать аватарки
- **Esc** — закрыть зум

## Хранение

Аватарки хранятся как сжатый base64 (JPEG, макс. 512px) в настройках расширения. Экспортируются вместе с бэкапом ST.
