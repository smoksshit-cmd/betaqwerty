# 🧠 Трекер памяти фактов (Facts Memory Tracker)

Расширение для SillyTavern, которое автоматически извлекает важные факты из истории чата и хранит их компактно — вместо тяжёлой истории в промпт попадает только нужное.

---

## Зачем это нужно

Длинные RP-чаты пожирают токены. Вместо того чтобы гнать в контекст 200 сообщений, FMT держит живую базу ключевых фактов и инжектирует их одним компактным блоком. Модель помнит важное, не теряет нить, а ты экономишь токены.

---

## Установка

1. Скопируй папку `facts-memory-tracker` в:
   ```
   SillyTavern/public/scripts/extensions/third-party/
   ```
2. Перезагрузи SillyTavern
3. Расширение появится в панели настроек — раздел **«🧠 Трекер памяти фактов»**

## Возможности

### 🔍 Извлечение фактов
- **Ручной скан** — нажми «Сканировать» и AI проанализирует историю чата, вытащит важные факты без дублей
- **Авто-скан** — срабатывает автоматически каждые N сообщений (настраивается)
- **Авто-маркер** — если модель пишет `[FACT: текст | категория]` в ответе, факт добавляется мгновенно без скана

### 📂 Категории фактов
| Категория | Что хранит |
|-----------|-----------|
| 👤 Персонажи | Имена, роли, черты, отношения между персонажами |
| 📅 События | Ключевые события, решения, их последствия |
| 🔒 Секреты | Тайны, скрытые мотивы, компромат |
| 🌀 Флешбэки | Воспоминания персонажа, травмы, образы из прошлого |

### 💉 Инъекция в промпт
Факты инжектируются компактным блоком:
```
[ПАМЯТЬ ФАКТОВ]
Ключевые факты о мире, персонажах и событиях этого RP:
👤 ПЕРСОНАЖИ: Алиса — детектив, скрывает прошлое | Виктор — информатор, боится мести
📅 СОБЫТИЯ: Алиса нашла улику в подвале | Виктор передал конверт с данными
🔒 СЕКРЕТЫ: Алиса работала на картель три года назад
[/ПАМЯТЬ ФАКТОВ]
```
- Настраиваемый порог важности (все / medium+ / только high)
- Лимит фактов в инъекции (5–100)
- Кастомный шаблон блока с плейсхолдером `{{facts}}`

### ⚡ Флешбек-триггер
Заставляет персонажа внезапно вспомнить факт из прошлого прямо в следующем ответе — через образ, паузу, реакцию, не разрывая сцену.

- **Вручную** — кнопка «⚡ Флешбек» в трекере или кнопка ⚡ на конкретном факте
- **Авто-шанс** — задай процент (0–30%) и флешбеки будут случайно прорываться сами
- Выбор категорий для выборки (воспоминания, секреты, персонажи)

### ✏️ Управление фактами
- **Inline-редактирование** — кликни на текст факта прямо в списке
- **Смена категории и важности** — через дропдауны в строке
- **Отключить без удаления** — кнопка ⏸ скрывает факт из инъекции, но сохраняет его
- **Поиск** по тексту, сортировка по дате / важности / категории
- **Сворачиваемые категории** — клик на заголовок секции

### 📤 Экспорт / Импорт
- Экспорт фактов в JSON с кнопкой «Скопировать»
- Импорт JSON — перенос фактов между чатами
- Лог сканирований — история когда и сколько фактов было найдено

### 🔢 Счётчик токенов
В трекере показывается приблизительное количество токенов инжектируемого блока в реальном времени.

## Интерфейс

**Виджет 🧠** — плавающая кнопка, показывает количество активных фактов. Полупрозрачная пока не наведёшь. Перетаскивается мышью. Размер регулируется слайдером в настройках (40–140%).

**Трекер** — боковая панель со всеми фактами, фильтрами, поиском и кнопками управления.

**Настройки** — компактный аккордеон из 5 секций, каждая сворачивается отдельно:
- ⚙️ Основное
- 🔍 Сканирование
- 💉 Инъекция
- ⚡ Флешбек
- 🔌 API

---

## API для сканирования

По умолчанию используется встроенный ST (`generateRaw`). Для лучшего качества можно подключить внешний API:

1. Укажи **Endpoint** (например `https://api.openai.com/v1`)
2. Укажи **API Key**
3. Нажми 🔄 для загрузки списка моделей
4. Выбери модель

Поддерживаются любые OpenAI-совместимые прокси.

---

## Авто-маркер [FACT:]

Добавь в системный промпт инструкцию для модели, чтобы она сама сигнализировала о важных фактах:

```
Когда в сцене появляется важный новый факт о персонаже, событии или секрете,
добавь в конец ответа маркер: [FACT: текст факта | категория]
Категории: characters, events, secrets, flashbacks
```

Расширение автоматически поймает маркер и добавит факт без скана.
